Fishing Chronicles
It is a fishing simulator. Where a player will start on the main screen and be able to enter the
game. Once they have started then they will spawn into the fishing scene and be able to play.
The objective is to catch and sell fish, to then spend the money earned on better fish, then repeat
the process.

Build:
Using unity, you use unity’s ability to build projects.
